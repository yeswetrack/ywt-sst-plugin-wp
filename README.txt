YesWeTrack server side tracking plugin for woocommerce

== Installation ==

1. Unzip and upload the "yeswetrack-sst" to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings / YesWeTrack and enter your YesWeTrack SST url and set additional options
